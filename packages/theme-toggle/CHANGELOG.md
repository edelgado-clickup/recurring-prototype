# Changelog

## [1.0.0] - 2025-12-10

### Added
- Initial release of Theme Toggle component
- Angular standalone component for light/dark mode switching
- SVG icons embedded directly in the component (no external assets needed)
- Automatic `dark-theme` class management on document root
- Positioned at top-left (24px, 24px) by default
- Hover states for both light and dark modes
- Full TypeScript support
- Comprehensive documentation and usage guides
